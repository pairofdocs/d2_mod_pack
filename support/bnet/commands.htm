    COMMANDES AVANCÉES

    Lorsque vous utilisez les commandes avancées, vous pouvez également vous servir du [nom de personnage] de l'utilisateur pour les personnes dans le même royaume et du [nom de personnage]@Realm pour les personnes des autres royaumes.

    Vous devez utiliser les *noms de compte pour toutes les personnes se trouvant sur Battle.net, y compris les personnes jouant des parties différentes.

    A chaque fois que les commandes décrites ci-dessous nécessite l'utilisation d'un [nom de compte], veuillez utiliser soit [nomdepersonnage], soit [nomdepersonnage@Realm], soit [*nomdecompte].

    Les commandes suivantes peuvent être utilisées dans n'importe quelle chaîne de discussion. Elles sont également disponibles lors de la configuration d'une partie, ainsi qu'en cours de partie.

    /help - (Ex : /help whisper)
    Affiche une aide utilisateur sur les fonctions de discussion.

    /whois [utilisateur] - (Ex : /whois TPark or /whois #133115)
    Recherche les informations essentielles concernant un utilisateur, y compris sa position actuelle. Un bon moyen de retrouver des amis actuellement connectés. Syntaxe : /whois #[numéro] (par ex. /whois #1).
    Autres noms de cette commande : /where /whereis

    /whoami - (Ex : /whoami) - affiche les informations vous concernant sur Battle.net.

    /whisper [utilisateur] - (Ex : /whisper JohnS ou /whisper #236325)
    Adresse un message personnel à un utilisateur connecté, quelle que soit la chaîne dans laquelle il se trouve. Un bon moyen de faire savoir à un ami que vous êtes en ligne.
    Autre nom de cette commande : /w /msg /m

    /squelch [utilisateur] - (Ex : /squelch BFitch ou /squelch #223251)
    Ignore tous les messages émis par cet utilisateur, qui devient ainsi muet.
    Autre nom de cette commande : /ignore

    /unsquelch [user] - (Ex : /unsquelch BFitch ou /unsquelch #223251)
    Permet à un utilisateur rendu "muet" (commande squelch) de discuter à nouveau normalement avec vous.
    Autre nom de cette commande : /unignore

    /away [texte] - (Ex : /away suis parti manger)
    Renvoie un message indiquant que vous n'êtes pas devant votre ordinateur à tout utilisateur vous chuchotant un message (commande "whisper") ou ayant saisi une commande /whois à votre sujet. Pour désactiver cette commande, utilisez-la à nouveau.

    /dnd [texte] - (Ex : /dnd désolé je fais une partie avec GFraizer)
    "Do Not Disturb" (Ne pas déranger) - empêche l'affichage sur votre écran de TOUS les messages chuchotés (commande "whisper"). Utilisation semblable à la commande /away.
    Pour désactiver le mode DND, il vous suffit d'entrer la commande /dnd, sans indiquer de raison. Cette commande empêche également les joueurs de vous chuchoter des messages en cours de partie. Les messages normaux des chaînes publiques vous restent néanmoins accessibles.

    /who [chaîne] - (Ex : /who Open Tech Support)
    Affiche la liste de tous les utilisateurs présents dans la chaîne demandée.

    /stats [utilisateur] ID - (Ex : /stats DavidN STAR ou /stats #42 STAR)
    Affiche le journal des Victoires/Défaites/Déconnexions pour le joueur demandé, aussi bien pour les parties normales que les parties en mode Echelle.


      STAR - Starcraft
      SEXP - Extension Starcraft (Brood War)
      W2BNE - Warcraft II Battle.net Edition

    /users
    Affiche le nombre d'utilisateurs actuellement connectés à Battle.net.

    /time
    Affiche l'heure actuelle sur Battle.net.



    Les commandes suivantes ne peuvent être utilisées que dans un forum de discussion

    /channel [nom de chaîne] - (Ex : /channel Blizzard Tech Support)
    Permet de basculer directement dans une chaîne, sans devoir passer par l'écran correspondant.
    Autres noms de cette commande : /join

    /me [texte] - (Ex : /me je suis mort de rire !)
    Affiche un texte à valeur émotive, tel une phrase du type "PatN est mort de rire !".
    Autres noms de cette commande : /emote



    Les commandes suivantes ne peuvent être utilisées que par les opérateurs de chaînes

    /designate [utilisateur] - (Ex : /designate AlenL ou /designate #423262)
    Désigne [utilisateur] comme prochain opérateur de chaîne lorsque vous aurez décidé soit de démissionner soit de quitter la chaîne.
    Si l'opérateur quitte la chaîne après avoir désigné plusieurs personnes, la dernière personne nommée est placée aux commandes de la chaîne.

    /resign
    Démissionner du poste d'opérateur de chaîne et redevenir un utilisateur normal.
    Autre nom de cette commande : /rejoin

    /kick [utilisateur] - (Ex : /kick TwainM ou /kick #234226)
    Exclut un utilisateur de la chaîne. Cette commande tient en fait lieu d'avertissement, puisqu'un utilisateur exclu peut revenir à tout moment.

    /ban [utilisateur] - (Ex : /ban RobertB ou /ban #137389)
    Exclut un utilisateur d'une chaîne et l'empêche d'y revenir jusqu'à ce que l'opérateur actuel ait démissionné ou quitté la chaîne. Il s'agit de l'une des commandes les plus puissantes mises à disposition des opérateurs. Utilisez-la avec modération.

    /unban [utilisateur] - (Ex : /unban RobertB ou /unban #137389)
    Autorise un utilisateur banni à revenir dans une chaîne.



    Touches de raccourci

    CTRL-X
    Coupe le texte sélectionné

    CTRL-C
    Copie le texte sélectionné

    CTRL-V
    Colle le texte sélectionné

    CTRL-A
    Sélectionne tout le texte

    CTRL-N
    Diablo II : colle le nom que vous avez sélectionné

    ALT-N
    Diablo, Starcraft et Warcraft II BNE : colle le nom que vous avez sélectionné

    /d2notify
    Diablo II : active ou désactive les avis d'entrée/de sortie d'une chaîne de discussion

    ALT-V
    Diablo, Starcraft et Warcraft II BNE : active ou désactive les avis d'entrée/de sortie d'une chaîne de discussion

    ALT-W après avoir sélectionné un utilisateur
    Diablo, Starcraft, Warcraft II BNE : adresse un message personnel à l'utilisateur sélectionné
    Autre nom de cette commande : /w /msg /m /whisper

    TAB
    Diablo II : fait défiler les 10 dernières commandes utilisées
    Starcraft : affiche la dernière commande utilisée, puis fait défiler les noms des utilisateurs présents dans la chaîne actuelle

    CTRL-M
    Diablo II : active ou désactive la musique sur Battle.net et dans les menus principaux
