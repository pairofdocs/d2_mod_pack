    • Si le problème persiste, supprimez le fichier BNCACHE.DAT qui se trouve dans le dossier dans lequel vous avez installé le jeu. Si celui-ci est corrompu, il est susceptible d'être à l'origine du problème.
    • Ensuite, supprimez le contenu du dossier "TEMP" de votre répertoire Windows (habituellement c:\windows\temp).
    • Consultez la page des mises à jour de Windows afin de vous assurer que vous disposez bien des dernières mises à jour de Windows.
