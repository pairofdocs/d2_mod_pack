    Si vous recevez le message "Battle.net n'a pas pu identifier correctement la version de votre application" ou Patcher.cpp ligne 462 :

      Ce problème est généralement provoqué par un virus nommé "Windows CIH virus".
      Pour éradiquer le virus et résoudre le problème relatif à ce dernier, veuillez suivre la procédure suivante :

      Désinstallez le jeu

      • Cliquez sur "Démarrer"
      • Sélectionnez "Programmes"
      • Sélectionnez le jeu que vous souhaitez désinstaller
      • Cliquez sur "Désinstaller"

      Eradication du virus

        Les liens suivants vous amèneront sur les sites de sociétés produisant des logiciels antivirus.
        • http://www.avp.ch/E/faq/cihfaq.htm (Win95.CIH.1019)
        • http://www.symantec.com/avcenter/data/cih.html (W95.CIH)
        Suivez les instructions fournies par le fabricant du logiciel antivirus afin de détecter et détruire les virus présents sur votre système.

      Réinstallez le jeu

      • Insérez le CD et cliquez sur Installer. Si l'écran d'exécution automatique ne s'affiche pas :
        • Double-cliquez sur l'icône "Poste de travail"
        • Double-cliquez sur l'icône du CD
        • Double-cliquez sur l'icône "Setup"
      • Le programme d'installation vous fournira ensuite les indications nécessaires à l'installation du jeu.
