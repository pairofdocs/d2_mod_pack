    Mises à jour de Windows
      Mises à jour de sécurité Microsoft :
      • Consultez le site internet de Microsoft à l'adresse http://www.microsoft.com/security/default.asp pour obtenir de nouveaux correctifs de sécurité.

      Pour déterminer quelle version de Windows vous utilisez :

      • Cliquez sur "Démarrer"
      • Cliquez sur "Paramètres"
      • Cliquez sur "Panneau de configuration"
      • Double-cliquez sur l'icône "Système"
      • La version de Windows devrait figurer sous le nom Microsoft Windows.

      Toutes versions de Windows :

      • Consultez la page des mises à jour de Windows afin d'obtenir d'éventuelles mises à jour de votre système d'exploitation.

      Toutes versions de Windows 95 :

      • Regclean est un programme conçu pour nettoyer les entrées de registre inutiles.
      Correctifs de sécurité permettant de répondre à quelques attaques communes
      • Attaques Out-of-Band, allez à la page de recherche Microsoft et effectuez une recherche sur "Update to Windows 95 TCP/IP to Address Out-of-Band Issue".
      • Attaques Corrupted Internet Control Message Protocol (ICMP) datagram fragment, allez à la page de recherche Microsoft et effectuez une recherche sur "Invalid ICMP Datagram Fragments Hang Windows NT, Windows 95".

    Windows NT :

    • Les utilisateurs de Windows NT doivent s'assurer d'avoir installé la dernière version du Service Pack for NT.
    • Nous leur recommandons également d'installer tous les correctifs Microsoft pouvant s'appliquer à leur ordinateur.
    • Assurez-vous d'avoir installé toutes les correctifs de sécurité afin d'éviter que des utilisateurs distants ne puissent bloquer votre système via une connexion Internet.
