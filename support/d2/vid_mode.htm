    Comment changer le mode vidéo utilisé dans l'Extension de Diablo II :

      Depuis le bureau de Windows, cliquez sur Démarrer --> Programmes --> Diablo II Extension --> Test Vidéo.
      Le programme teste la configuration de votre ordinateur et vous propose de choisir un mode vidéo.


      Si le mode vidéo choisit ne fonctionne pas correctement, contactez le fabricant de votre carte graphique pour obtenir les pilotes les plus récents.


      L'Extension de Diablo II est compatible Glide™ et Direct 3D™ si votre carte dispose d'au moins 8 Mo de RAM vidéo. Le mode Direct 3D™ nécessite que vous disposiez d'au moins 64 Mo de RAM.
      La résolution par défaut de l'écran de jeu dans Diablo II Extension est de 640 x 480. Lorsque vous vous connectez sur Battle.net avec Diablo II: Lord of Destruction, la résolution utilisée est de 800 x 600.
