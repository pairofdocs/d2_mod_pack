    Le jeu est lent ou n'est pas fluide :

    Si vous avez l'impression que votre jeu accélère après une période de ralentissement voire d'arrêt complet, essayez les étapes suivantes.

    Fermez tous les programmes résidents :

    Fermer des programmes résidents sous Windows 95/98

    1. Appuyez simultanément sur les touches <Ctrl>, <Alt> et <Suppr>. La fenêtre Fermer le programme apparaît.
    2. Sélectionnez tous les programmes (sauf Explorer et Systray) et cliquez sur le bouton "Fin de tâche".
      Remarque : ces programmes seront automatiquement relancés lorsque vous redémarrez votre ordinateur.
    Fermer des programmes accessoires sous Windows NT/2000:
    1. Appuyez simultanément sur les touches <Ctrl>, <Alt> et <Suppr> pour afficher un écran de sécurité.
    2. Appuyez sur le bouton des tâches pour accéder à une fenêtre décrivant tous les programmes actifs.
    3. Sélectionnez un programme et cliquez sur "Fin de tâche". Répétez les deux premières étapes pour afficher de nouveau la liste et continuez ainsi jusqu'à ce que vous ayez fermé tous les programmes.
      Remarque : si votre système bloque ou redémarre, vous devez contacter le fabricant de votre ordinateur pour apprendre comment fermer ces programmes en toute sécurité. Si votre système s'arrête, coupez l'alimentation, attendez 10 secondes puis rallumez.

    ***Ni Explorer ni Systray n'apparaîtront comme des tâches actives. Cette liste ne montre que les applications en cours d'utilisation qui ont ouvert une fenêtre (Visible). Tous les programmes qui fonctionnent en tâche de fond, y compris les logiciels comme les antivirus et les outils réseau, seront décrits dans la liste. Fermer les programmes antivirus améliore toujours considérablement les performances de la machine, mais n'oubliez pas de les activer de nouveau lorsque vous avez fini de jouer !

    Vérifiez la carte réseau et ses pilotes.

    Rendez-vous sur le site Internet du fabricant de votre carte réseau et vérifiez que vous utilisez les pilotes les plus récents pour votre carte réseau. Assurez-vous également que votre carte réseau est bien fixée dans son emplacement. Les cartes mal fixées peuvent causer des problèmes de communication et occasionner des pauses ou des ralentissements du jeu.

    Régler l'ordinateur sur Serveur Réseau :

    1. Cliquez avec le bouton droit de la souris sur Poste de travail.
    2. Sélectionnez propriétés.
    3. Cliquez sur l'onglet Performance.
    4. Cliquez sur le bouton Système de fichiers.
    5. Réglez l'utilisation typique de votre ordinateur sur Serveur réseau.
    6. Cliquez sur OK et redémarrez votre ordinateur.

    Si vous êtes actuellement en mode 3D, passez en mode 2D.

    1. Cliquez sur Démarrer>Programmes>Diablo II Extension>Test vidéo. Lancez le text vidéo et sélectionnez l'option vidéo 2D.

    Lancez Scandisk et Defrag

    Si vous avez rencontré des problèmes sous Windows et/ou des problèmes de disque dur, lancez Scandisk et Defrag. Il est conseillé d'utiliser régulièrement ces logiciels pour conserver des performances optimales.

    Pour lancer Scandisk :

    • Cliquez sur Démarrage, Programmes, Accessoires et Outils système.
    • Cliquez sur l'icône Scandisk.
    • Validez Standard pour un test rapide, ou Minutieuse pour un test complet beaucoup plus long.
    • Cochez la case Réparer automatiquement les erreurs.
    • Sélectionnez le disque dur à tester et cliquez sur Démarrer.

    Pour lancer Defrag:

    • Cliquez sur Démarrage, Programmes, Accessoires et Outils système.
    • Cliquez sur l'icône Defrag.
    • Sélectionnez le disque dur à défragmenter.
    • Cliquez sur Démarrer.

    Vérifiez les paramètres de Mémoire Virtuelle

    Définissez une grande taille de mémoire virtuelle si vous disposez de suffisamment d'espace libre sur le disque (assurez-vous que les tailles minimales et maximales sont les mêmes).

    • Nous vous recommandons d'avoir au moins 350 Mo d'espace libre sur votre disque dur.
    • N'oubliez pas de lancer Defrag & Scandisk avant de changer les réglages de votre Mémoire Virtuelle.

      Windows 95/98 :

      1. Cliquez avec le bouton droit de la souris sur Poste de travail.
      2. Cliquez sur Propriétés.
      3. Cliquez sur l'onglet Performances.
      4. Cliquez sur Mémoire virtuelle.
      5. Cochez "Me permettre de spécifier mes propres paramètres de mémoire virtuelle".
      6. Indiquez des tailles minimale et maximale telles qu'indiquées après selon la RAM dont vous disposez.
        Par exemple :
        • 32-128 Mo de RAM : Min 300, Max 300.
        • 192 Mo de RAM : Min 384, Max 384.
        • 256 Mo de RAM : Min 512, Max 512.
      7. Cliquez sur "OK" pour appliquer les changements.
      8. Faites redémarrer votre ordinateur.

      Windows 2000 :

      1. Cliquez avec le bouton droit de la souris sur "Poste de travail" pour accéder aux "Propriétés système"
      2. Cliquez sur Avancés
      3. Cliquez sur Options performances... pour afficher l'écran idoine.
      4. Cliquez sur "Changer..." pour afficher l'écran "Mémoire virtuelle".
      5. Indiquez des tailles minimale et maximale au double de la RAM dont vous disposez.
        Par exemple :
        • 32-128 Mo de RAM : Min 300, Max 300.
        • 192 Mo de RAM : Min 384, Max 384.
        • 256 Mo de RAM : Min 512, Max 512.
      6. Cliquez sur "OK" pour appliquer les changements.
      7. Faites redémarrer votre ordinateur.

      Windows NT

      1. Cliquez avec le bouton droit de la souris sur "Poste de travail" pour accéder aux "Propriétés système"
      2. Cliquez sur Options performances... pour afficher l'écran idoine.
      3. Cliquez sur "Changer..." pour afficher l'écran "Mémoire virtuelle".
      4. Indiquez des tailles minimale et maximale au double de la RAM dont vous disposez.
        Par exemple :
        • 32-128 Mo de RAM : Min 300, Max 300.
        • 192 Mo de RAM : Min 384, Max 384.
        • 256 Mo de RAM : Min 512, Max 512.
      5. Cliquez sur "OK" pour appliquer les changements.
      6. Faites redémarrer votre ordinateur.

      Si vous utilisez actuellement le mode plein écran, passez en mode fenêtré.

      1. Cliquez avec le bouton droit de la souris sur le bouton Démarrer.
      2. Cliquez sur Programmes>Diablo II Extension.
      3. Cliquez avec le bouton droit de la souris sur le raccourci Diablo II Extension et sélectionnez Propriétés.
      4. Dans le champ cible, ajouter un " -w" après les guillemets (i.e., "C:\Program Files\Diablo II Expansion\Diablo II Expansion.exe" -w ).
      5. Cliquez sur OK.

      Assurez-vous que vous possédez la dernière version du bios de votre carte-mère.

      Si vous avez un faible taux de rafraîchissement et si vous souffrez d'un manque de fluidité lorsque vous jouez à l'Extension de Diablo II, assurez-vous que vous possédez la dernière version du bios de votre carte-mère. Elle vous est souvent livrée sur disquette avec la carte-mère, mais vous devriez également le télécharger sur le site Web du fabricant.

      Vérifiez si vous avez une carte mère VIA.

      Les utilisateurs de cartes mères VIA devraient télécharger la mise à jour de pilote "4-in-1", qui détecte et met à jour automatiquement les paramètres IRQ, AGP, IDE et INF. Cela devrait améliorer la fluidité du jeu sur les systèmes performants et empêcher les plantages sur les cartes mères VIA.

      Copiez les fichiers de jeu sur le disque dur.

      Si vous continuez à souffrir de manque de fluidité et de lenteur, vous pouvez copier les fichiers D2CHAR.MPQ et D2MUSIC.MPQ du disque de Diablo II Expansion jusque dans le répertoire Diablo II Expansion sur votre disque dur. Cela devrait réduire les temps d'accès au CD et de lecture.
