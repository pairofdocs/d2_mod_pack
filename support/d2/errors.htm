    Messages d'erreur au lancement de Diablo II: Lord of Destruction :

      error 1 : Diablo II: Lord of Destruction is unable to proceed. Please run D2VidTst and try again (Diablo II: Lord of Destruction ne peut s'exécuter correctment. Veuillez lancer le programme D2VidTst, puis réessayer d'exécuter le jeu).

      error 21 : A critical error has occurred while initializing windowed mode (Une erreur critique est survenue à l'initialisation du mode paginé).

      error 22 : A critical error has occurred while initializing DirectDraw (Une erreur critique est survenue à l'initialisation de DirectDraw).

      error 23 : A critical error has occurred while initializing Glide (Une erreur critique est survenue à l'initialisation de Glide).

      error 24 : A critical error has occurred while initializing OpenGL (Une erreur critique est survenue à l'initialisation de OpenGL).

      error 25 : A critical error has occurred while initializing Direct3D (Une erreur critique est survenue à l'initialisation de Direct3D).



    Dépannage :

    • Relancez le programme de Test vidéo de Diablo II: Lord of Destruction.
    • Téléchargez et installez les pilotes vidéo disponibles sur le site de votre constructeur de carte vidéo ou d'ordinateur.
    • Si les problèmes persistent, lancez le programme de Test vidéo et réglez le mode vidéo sur DirectDraw.
