    Diablo II: Lord of Destruction se ferme continuellement et me renvoie sous Windows :

      Certains logiciels peuvent provoquer la réduction de l'application Diablo II: Lord of Destruction ou peuvent empiéter sur l'écran du jeu. Pour remédier à cela, fermez toutes les applications ouvertes avant de lancer Diablo II: Lord of Destruction.


    Etapes à suivre pour fermer les applications fonctionnant en tâche de fond :

      Appuyez simultanément sur les touches Ctrl, Alt et Suppr. La fenêtre Fermer le programme apparaît. Sélectionnez Fin de tâche pour tous les programmes à l'exception de Explorer et Systray.
      Remarque : ces programmes se lanceront à nouveau lorsque vous redémarrerez votre ordinateur.


    Etapes à suivre pour modifier vos fichiers de démarrage :

    • Cliquez sur "Démarrer", choisissez "Exécuter...", tapez"msconfig" et appuyez sur la touche Entrée.
    • Sélectionnez l'onglet Démarrage
    • Assurez-vous que toutes les cases à cocher sont vides, puis cliquez sur OK.
    • Lorsque la fenêtre "Modifier les paramètres système" s'affiche, cliquez sur OK pour redémarrer l'ordinateur.
      Remarque : pour redémarrer ces applications, il suffit de lancer "msconfig", cocher les cases et redémarrer votre ordinateur.
