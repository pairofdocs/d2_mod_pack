Informations Système Microsoft (msinfo32)


    Windows 95 :
    • Assurez-vous que le CD de Diablo II se trouve bien dans votre lecteur CD-ROM
    • Cliquez sur Démarrer, Rechercher, Fichiers ou Dossiers
    • Dans la zone "Nommé", saisissez "msinfo32.exe"
    • Dans la zone "Rechercher dans", sélectionnez "Lecteur C (C:)" ou "Disques durs locaux (x:, x:, ...)"
    • Cliquez sur Rechercher pour rechercher le fichier, ou double-cliquez sur le fichier pour l'exécuter
    • Si vous ne trouvez pas le fichier msinfo32.exe :
      • Ouvrez n'importe quel dossier, Affichage, Options ou Options des dossiers, et cliquez sur l'onglet Affichage
      • L'option "Masquer les extensions dont le type de fichier est connu" ne devrait pas être activée
      • L'option "Afficher tous les fichiers" devrait être cochée
      • Cliquez sur Appliquer puis sur OK, puis effectuez à nouveau une recherche sur msinfo32.exe
    • Cliquez sur Fichier, Enregistrer
    • Vous verrez s'afficher une boîte de dialogue indiquant à quel emplacement le fichier texte a été enregistré, et sous quel nom.
    • Joignez ce fichier texte à un e-mail adressé à support@blizzard.com en fournissant les informations requises sur la page contact.

    Windows 98 :

    • Assurez-vous que le CD de Diablo II se trouve bien dans votre lecteur CD-ROM
    • Cliquez sur Démarrer, Programmes, Accessoires, Outils système, Informations système.
    • Sélectionnez Fichier, puis Exporter...
    • Joignez ce fichier texte à un e-mail adressé à support@blizzard.com en fournissant les informations requises sur la page contact.

    Windows 2000 :

    • Assurez-vous que le CD de Diablo II se trouve bien dans votre lecteur CD-ROM
    • Cliquez sur Démarrer, et sélectionnez Exécuter
    • Sélectionnez Exécuter
    • Entrez : msinfo32
    • Cliquez sur OK
    • Sélectionnez Action
    • Enregistrer au format texte...
    • Entrez le nom du fichier, puis sélectionnez Enregister
    • Joignez ce fichier texte à un e-mail adressé à support@blizzard.com en fournissant les informations demandées sur la page contact.
