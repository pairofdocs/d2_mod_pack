    La plupart des problèmes vidéo ou audio peuvent être résolus en suivant ces étapes :
    • Contactez le fabricant de votre ordinateur ou de vos cartes vidéo et son (si vous les avez achetées séparément) afin de vous informer sur les versions de DirectX compatibles avec leurs pilotes.
    • Téléchargez et installez la version de Direct X recommandée par votre fabricant.
    • Exécutez le programme de test vidéo et utilisez les paramètres par défaut.
    • Si les problèmes persistent, exécutez le programme de test vidéo et sélectionnez le mode DirectDraw.

    Exécutez le programme de test vidéo afin de sélectionner un mode vidéo :

    • Cliquez sur Démarrer
    • Sélectionnez Programmes
    • Sélectionnez Diablo II: Lord of Destruction
    • Cliquez sur Test vidéo

    Modes vidéo supportés par Diablo II: Lord of Destruction :

      Diablo II: Lord of Destruction supporte les cartes graphiques compatibles Glide™ et Direct 3D™ disposant d'au moins 8 Mo de RAM vidéo.
      Direct 3D™ requiert 64 Mo de RAM système.
      L'interface de Diablo II: Lord of Destruction est en 640x480, tandis que l'interface de discussion de Battle.net fonctionne avec une résolution de 800x600.

    Son instable :

    • Cliquez sur Démarrer, Paramètres, Panneau de configuration, puis double-cliquez sur Multimédia.
    • Assurez-vous que votre carte audio est bien définie en Périphérique par défaut.
    • Assurez-vous que la case "Utiliser uniquement les périphériques par défaut" est cochée.

    Les sons ne correspondent pas aux animations des personnages dans le menu de choix d'une catégorie :

    • Cliquez sur Démarrer, Paramètres, Panneau de configuration, puis double-cliquez sur l'icône Multimédia.
    • Assurez-vous que la case "Utiliser uniquement les périphériques par défaut" est cochée.
    • Ce problème peut également être dû à un manque de RAM.
